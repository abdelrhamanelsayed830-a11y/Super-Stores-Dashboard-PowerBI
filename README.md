# 📊 Super Stores Analysis Dashboard - Power BI

## 📌 Overview
Interactive Super Stores Dashboard built from scratch using real sales data.

## 🛠️ Tools Used
- Power BI Desktop
- DAX (Custom Measures)
- Data Modeling

## 🔑 Key Features
- KPI Cards: Total Sales, Total Profit, Total Orders, Total Quantity & AVG-Discount
- Orders by Region
- Orders Trend by Month
- Orders by Sub-Category
- Orders by Category (Pie Chart)
- Map Page for Geographic Analysis
- Interactive Slicer: Year

## 💡 Key Insights
- Office Supplies dominates with 53% of orders
- West region leads with 3.2K orders
- Orders peak toward end of year (November & December)
